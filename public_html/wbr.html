Exemplo

<p>Para aprender AJAX, você primeiro deve estar familiar com o XML<wbr>Http<wbr>Request Object.</p>

Definição e Uso

The <wbr> (Word Break Opportunity) tag specifies where in a text it would be ok to add a line-break.
