<span>

Exemplo

Um span que é usado para colorir uma parte de um texto:

My mother has <span style="color:blue">blue</span> eyes.

Definição e Uso

A tag <span> A tag é um recipiente interno usado para marcar uma parte de um texto ou uma parte de um documento.
A tag é facilmente estilizada por CSS ou manipulada com JavaScript usando o atributo de classe ou id.
A tag é muito parecida com o <div> element, mas <div> é um elemento de nível de bloco e é um elemento inline.


Atributos Globais

A tag também suporta os Atributos Gloabis HTML.
