O que são HTML e CSS?

HTML5

HTML5 é uma sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Ela traz novas funções como as tags <video> , <audio> e elemento <canvas> e será o novo padrão para HTML, XHTML e HTML DOM.

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.
