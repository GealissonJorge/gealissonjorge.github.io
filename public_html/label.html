Definição e Uso

A tag <label> A tag define um rótulo para vários elementos:
<input tipo="caixa de seleção" >
<input tipo="cor">
<input tipo="data">
<input tipo="datetime-local">
<input tipo="e-mail">
<input type="arquivo">
<input tipo="mês">
<input tipo="número">
<input tipo="senha" >
<input tipo="rádio">
<input type="range">
<input tipo="pesquisa">
<input tipo="tel">
<input tipo="texto">
<input tipo="tempo">
<input type="url">
<input tipo="semana">
<meter>
<progress>
< select>
<textarea>
O uso adequado de etiquetas com os elementos acima beneficiará:
Usuários leitores de tela (lerão em voz alta o rótulo, quando o usuário estiver focado no elemento);
Usuários que têm dificuldade em clicar em regiões muito pequenas (como caixas de seleção) - porque quando um usuário clica no texto dentro do elemento, ele alterna a entrada (isso aumenta a área de impacto).

Atributos

Atributo Descrição
for Especifica o id do elemento do formulário ao qual o rótulo deve ser vinculado.
form Especifica a qual formulário o rótulo pertence.
