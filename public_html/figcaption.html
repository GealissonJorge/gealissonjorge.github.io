<figcaption>

Exemplo

Use um elemento <figure>se para marcar uma foto em um documento, e um elemento <figcaption> para definir uma legenda para a foto:

<figure>
<img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
<figcaption> Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Definição e Uso

A tag <figcaption> A tag define uma legenda para um <figure> element.
O elemento pode ser colocado como o primeiro ou último filho do <figure>element.


Atributos Globais

A tag também suporta os Atributos Gloabis HTML.
