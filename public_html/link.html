<link>

Exemplo

Definindo um link de estilos externo:

<head>
  <link rel="stylesheet" href="styles.css">
</head>

Definição e Uso

O<link> define um relacionamento entre a folha corrente e uma folha externa.
O<link>é mais usado para linkar uma folha de estilos.
O<link>é um elemento vazio e contém apenas atributos.

Atributos

Atributo Value Descrição
crossorigin anonymous use-credentials Especifica como o elemento lida com solicitações de origem cruzada
href URL Especifica a localizão do documento
hreflang language_code Especifica a linguagem do texto no documento linkado
media media_query Especifica em qual dispositivo o documento vinculado será exibido
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
unsafe-url 		Especifica qual referenciador usar ao buscar o recurso
rel alternate
author
dns-prefetch
help
icon
license
next
pingback
preconnect
prefetch
preload
prerender

search
stylesheet		 Obrigatório. Especifica a relação entre o documento atual e o documento vinculado
sizes Height X Width Especifica o tamanho do recurso vinculado. Apenas para o rel="icon"
title Define uma folha de estilo preferida ou alternativa
type media_type Especifica um tipo de media no recurso linkado
