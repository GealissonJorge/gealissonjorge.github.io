<option>

Exemplo

Uma lista suspensa com quatro opções:

<label for="cars"> Choose a car:</label>
<select id="cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="opel">Opel</option>
<option value="audi">Audi</option>
</select>

Definição e Uso

A tag <option> define uma opção em uma lista de seleção.
<option>elementos vão dentro de um elemento <select> , <optgroup> ou <datalist> .
Observação: a <option>tag pode ser usada sem nenhum atributo, mas geralmente você precisa do atributo value , que indica o que é enviado ao servidor no envio do formulário.
Dica: Se você tiver uma longa lista de opções, poderá agrupar opções relacionadas na tag <optgroup> .

Atributos

Atributo Descrição
disabled Especifica que uma opção deve ser desativada
label Especifica um rótulo mais curto para uma opção
selected Especifica que uma opção deve ser pré-selecionada quando a página carrega
value Especifica o valor a ser enviado para um servidor
