Referencias de Elementos HTML

Tag Descrição
<wbr> Representa a posição onde o navegador pode quebrar uma linha, opcionalmente, mesmo que suas regras digam do contrário.
<option> Indica que uma opção deve ser pré-selecionada quando a página é carregada
<code> Indica que um pedaço de código está sendo usado.
<i> Indica que uma parte de um texto é destecado por itálico.
<div> Indica uma divisão ou uma seção na página HTML.
<sub> Indica um parte do texto que é subscrito.
