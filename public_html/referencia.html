Referências de Elementos HTML

Tag Descrição
<audio> Tag usada para reproduzir um audio em página Web.
<button> Define um botão clicável.
<code> Indica que um pedaço de código está sendo usado.
<dialog> Tag usada para abrir uma janela de dialogo.
<div> Indica uma divisão ou uma seção na página HTML.
<figcaption> Define uma legenda para uma figura.
<i> Indica que uma parte de um texto é destecado por itálico.
<label> Define um rótulo para uma entrada.
<legend> Define uma legenda para um fieldset.
<option> Indica que uma opção deve ser pré-selecionada quando a página é carregada
<small> Define um texto menor.
<span> Define uma seção em um documento.
<strong> Define uma parte importante do texto no documento
<sub> Indica um parte do texto que é subscrito.
<wbr> Representa a posição onde o navegador pode quebrar uma linha, opcionalmente, mesmo que suas regras digam do contrário.
