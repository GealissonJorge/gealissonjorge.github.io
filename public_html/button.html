<button>

Exemplo

Um botão clicável é marcado da seguinte forma:

<button type="button">Click Me!<button>.

Definição e Uso

A tag <button> A tag define um botão clicável.
Dentro de um elemento você pode colocar texto (e tags como, , , , , , , etc.). Isso não é possível com um botão criado com os elementos <button><i><b><strong><br><input>.

Atributos

Atributo Descrição
autofocus Especifica que um botão deve ser automaticamente focado quando a página é carregada.
disabled Especifica que um botão deve ser desativado.
form Especifica a qual formulário o botão pertence.
formaction Especifica para onde enviar os dados do formulário quando um formulário é enviado. Apenas para type = "enviar".
formenctype Especifica como os dados do formulário devem ser codificados antes de enviá-los a um servidor. Apenas para type = "enviar".
formmethod Especifica como enviar os dados do formulário (qual método HTTP usar). Apenas para type = "enviar".
formnovalidate Especifica que os dados do formulário não devem ser validados no envio. Apenas para type = "enviar".
formtarget Especifica onde exibir a resposta após o envio do formulário. Apenas para type = "enviar".
name Especifica um nome para o botão.
type Especifica o tipo de botão.
value Especifica um valor inicial para o botão.
