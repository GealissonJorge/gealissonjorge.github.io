<s>

Exemplo

Marca um texto que não é mais correto:

<p><s>Only 50 tickets left!</s></p>.

Definição e Uso

A tag <s> especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha atravessada.
A tag <s> não deve ser usada para definir o texto excluído em um documento, use a tag <del> para isso.


Atributos Globais

A tag também suporta os Atributos Gloabis HTML.
