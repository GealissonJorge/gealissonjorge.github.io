<dialog>

Exemplo

Usando o elemento <dialog>:

<dialog open>This is an open dialog window<dialog>.

Definição e Uso

O <dialog> define uma caixa ou uma subjanela.
O <dialog> o elemento facilita a criação de diálogos pop-up e modais em páginas web.

Atributos

Atributo Value Descrição
open open Especifica que o elemento de diálogo está ativo e que o usuário pode interagir com ele
