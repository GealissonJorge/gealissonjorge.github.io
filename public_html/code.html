<code>

Exemplo

Defina algum texto como código de computador em um documento:

<p>A tag HTML <code>butão</code> define um butão clicável. </p>
<p>A propriedade CSS <code>background-color</code> define a cor de fundo de uma elemento. </p>

Definição e Uso

A tag <code> é usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
Dica: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS.
